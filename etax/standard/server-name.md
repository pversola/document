## Server Name

#### Format
| No | Detail | Lenght |
| :-: | - | :-: |
| 1 | Project Name | 3 |
| 2 | Site Name | 3 |
| 3 | Environment | 3 |
| 4 | Service | 2 |
| 5 | Running | 2 |

###### Example
 - Project: ETAX --> ETX
 - Site: NIPA CLOUD --> NPA | CSP | CSK | CSR
 - Environment: Develop --> DEV | UAT | PRODUCTION --> PRD
 - Serveice: Database --> DB | APPLICATION --> AP | 
 - Running: 1 --> 01
